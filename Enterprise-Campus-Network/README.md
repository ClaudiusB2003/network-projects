# Overview
This lab builds a realistic enterprise campus network featuring network segmentation, gateway redundancy, and perimeter security. It bridges CCNA-level fundamentals (VLANs, trunking, STP) with Security Basics (Firewalls, Policies, etc.).

## Topology
- 2x Distribution Switches
- 3x Access Switches
- 1x Firewall
- 1x Edge Router (internet connectivity)

## Technologies Used
- VLANs & 802.1Q Trunking
- Portchannel
- Switched Virtual Interfaces (SVIs)
- IPv4 / IPv6
- HSRP (First Hop Redundancy)
- Spanning Tree Root Bridge Design
- Inter-VLAN Routing
- OSPF
- DHCP
- Access Control Lists (ACLs)
- NAT (Network Address Translation)
- DNS
- Zone-Based Firewall
